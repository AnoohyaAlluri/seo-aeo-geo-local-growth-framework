# Internal Linking Map

## Purpose

This document explains the internal linking strategy used in the SEO, AEO, and GEO website growth framework.

Internal links help connect related pages, improve site structure, support search visibility, strengthen topical authority, and guide users toward conversion actions.

---

## Internal Linking Objective

The goal is to make every important page part of a clear content system.

A strong internal linking system should:

* Help users move from research to decision-making
* Connect city pages to service pages
* Connect educational content to conversion pages
* Support search-engine crawling
* Strengthen topical relationships
* Improve AI-readable site structure
* Reduce isolated or disconnected pages

---

## Core Linking Logic

```text
Homepage
      ↓
Core Service Pages
      ↓
City Pages
      ↓
Guides and FAQ Pages
      ↓
Conversion Pages
```

## Internal Link Types

| Link Type | Description | Example |
| --------- | ----------- | ------- |
| Service-to-City Link | Connects a core service page to city-specific pages | Property Management Services → Santa Monica Property Management |
| City-to-Service Link | Connects a city page back to core service details | Beverly Hills Property Management → Property Management Services |
| Guide-to-CTA Link | Moves informational users toward conversion | Cost Guide → Request Rental Analysis |
| FAQ-to-Service Link | Sends question-based users to a relevant service | Maintenance FAQ → Maintenance Coordination Services |
| Blog-to-Service Link | Connects educational content to business-service pages | Landlord Tips Blog → Property Management Services |
| City-to-Guide Link | Helps local visitors understand pricing or selection criteria | Brentwood Property Management → Cost Guide |

## Anchor Text Guidelines

Anchor text should be clear, natural, and specific.

Good examples:

* property management services
* property management cost guide
* Santa Monica property management
* maintenance coordination services
* owner FAQ hub
* schedule a consultation
* request a rental analysis

Avoid vague anchor text:

* click here
* learn more
* this page
* read more
* services

---

## Internal Linking Rules

1. Every service page should link to relevant city pages.
2. Every city page should link back to at least one core service page.
3. Every guide page should link to a conversion page or CTA.
4. Every FAQ page should link to relevant service pages.
5. Blog posts should support service pages, not remain isolated.
6. Priority city pages should receive more internal links than low-priority pages.
7. Anchor text should describe the destination page clearly.
8. Internal links should help users move logically through the site.
