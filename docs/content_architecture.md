# Content Architecture

## Purpose

This document explains the content architecture behind the SEO, AEO, and GEO website growth framework.

The goal is to organize website content so that each page has a clear purpose, search intent, internal-linking role, AI-answer function, and conversion path.

---

## Core Architecture Logic

The framework is built around this structure:

```text
Audience Need
      ↓
Search Intent
      ↓
Page Type
      ↓
Answer-First Section
      ↓
FAQ Support
      ↓
Schema Recommendation
      ↓
Internal Link Path
      ↓
Conversion CTA
